The Labseq Sequence

A recursive numerical pattern defined by its unique relationship to earlier terms. Unlike better-known sequences like Fibonacci that reference immediate predecessors, Labseq creates its rhythm by reaching further back in time.

Starting values

l(0) = 0

l(1) = 1

l(2) = 0

l(3) = 1

Recursive rule

l(n) = l(n-4) + l(n-3)

Sequence progression

0,1,1,2,3,4,5,7...
