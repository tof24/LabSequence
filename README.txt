LabSeqServer (Backend):
-----------------------
  - Open terminal in LabSeqServer directory.
  - Run: .\mvnw.cmd quarkus:dev
  - Access at: http://localhost:8080/labseq/n    // n=> A natural number 


Frontend:
-----------------------
  - Navigate to the Frontend directory.
  - Install dependencies if needed: npm install
  - Run: ng serve
  - Access at: http://localhost:4200
